FAQs

How do I upload files? To upload files you will need to make sure they are the following: PDF, JPG, JPEG, and PNG files. Continue to select the "Choose File"option to upload you files.

How can I save my files? Narrify will not save any files, they will be kept temporary to avoid storing information that can be issued in a copyright notice.

What do to if my text will not extract? Please ensure that you have the correct file format, clear image and clear text.
